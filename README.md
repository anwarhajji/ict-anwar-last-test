# ICT Anwar Trading Dashboard & Bot Manager

Ce projet est un tableau de bord de trading SaaS conçu pour les traders ICT (Futures/CFD), incluant un journal de trading, des analyses de performance, un calendrier économique, des checklists quotidiennes et un gestionnaire de bots de trading algorithmique.

## 🔗 Repository GitHub
[https://github.com/anwarhajji/ict-anwar-last-test.git](https://github.com/anwarhajji/ict-anwar-last-test.git)

## 🚀 Fonctionnalités
- **Journal de Trading :** Importation automatique (Tradovate) ou manuelle (CSV) des trades avec notes, captures d'écran et tags.
- **Analytics & Stats :** KPIs détaillés (Win Rate, Profit Factor, Expectancy, Drawdown) et courbe de capital.
- **Bots de Trading :** Création et déploiement de bots sur compte démo avec un éditeur de stratégie JavaScript.
- **Calendrier Économique :** Suivi des news à fort impact (High Impact News).
- **Checklists :** Tâches quotidiennes pré-market et post-market pour renforcer la discipline.
- **Profil & Paramètres :** Gestion du capital de départ et des limites de bots par utilisateur.

## 🛠️ Stack Technique
- **Frontend :** React, TypeScript, Tailwind CSS, Lightweight Charts.
- **Backend :** Firebase (Auth, Firestore), Express.js.
- **IA :** Intégration Gemini pour l'analyse des trades et la détection d'erreurs.

## 📦 Installation
1. Clonez le repository :
   ```bash
   git clone https://github.com/anwarhajji/ict-anwar-last-test.git
   ```
2. Installez les dépendances :
   ```bash
   npm install
   ```
3. Configurez les variables d'environnement dans un fichier `.env` (voir `.env.example`).
4. Lancez le serveur de développement :
   ```bash
   npm run dev
   ```
